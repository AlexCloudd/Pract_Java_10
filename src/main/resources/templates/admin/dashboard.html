О системе

AllIn - это современная система управления кинотеатром, построенная на микросервисной архитектуре с использованием Spring Boot.

  • Микросервисная архитектура
  • Spring Boot & Spring Cloud
  • PostgreSQL база данных
  • Eureka Service Discovery

Сервисы

Система состоит из нескольких микросервисов:

  • Movie Service (8083)
  • User Service (8082)
  • Rating Service (8084)
  • API Gateway (8080)
