Документация REST API для AllIn платформы

Обзор API

AllIn предоставляет RESTful API для доступа к функциональности платформы. Все API endpoints требуют базовый URL: http://localhost:8081/api

Основные Endpoints

Аутентификация

POST /api/auth/register
Регистрация нового пользователя в системе
POST /api/auth/login
Вход в систему и получение токена аутентификации
GET /api/auth/me
Получение информации о текущем аутентифицированном пользователе
POST /api/auth/logout
Выход из системы

Фильмы

GET /api/movies
Получение списка всех фильмов
GET /api/movies/{id}
Получение информации о фильме по ID
GET /api/movies/search?query={query}
Поиск фильмов по запросу
GET /api/omdb/popular
Получение популярных фильмов из OMDb API
GET /api/omdb/top-rated
Получение топ рейтинговых фильмов из OMDb API
GET /api/omdb/search?query={query}
Поиск фильмов в OMDb API
GET /api/omdb/movie/{imdbId}
Получение детальной информации о фильме по IMDb ID

Пользователи

GET /api/users
Получение списка всех пользователей
GET /api/users/{id}
Получение информации о пользователе по ID
GET /api/users/username/{username}
Получение информации о пользователе по имени пользователя

Списки просмотра

GET /api/watchlists
Получение списков просмотра текущего пользователя
POST /api/watchlists
Создание нового списка просмотра
